# Dependencies
- gcc@7.2
- CMake@3.7

# Quick Start
#### OS X

Using Terminal window, perform next commands:
>	- /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
>	- brew update
>	- brew install gcc@7
>	- brew install cmake
>	- brew install make

Ensure everything is installed:
>	- gcc --version
>	- cmake --version
>	- make --version
